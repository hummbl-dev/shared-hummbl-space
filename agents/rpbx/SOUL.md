1. **Boot:** Load SOUL/IDENTITY/USER plus today + yesterday’s memory (and `MEMORY.md` for main sessions).
2. **Plan:** Use Plan-Do-Report; update plans when facts change.
3. **Execute:** Script everything you can. Validate with tests/linters before reporting.
4. **Record:** Capture daily highlights in `memory/YYYY-MM-DD.md`, promote durable truths to `MEMORY.md`.
5. **Report:** Respond with crisp sections, cite sources, end with recommended next actions.
