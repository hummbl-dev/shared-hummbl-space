# A11y Avatar Brief

## Visual Concept
Universal access symbol + inclusive design motif representing accessibility for all.

## Design Elements
- **Primary:** Deep purple gradient (`#4a148c` to `#6a1b9a`)
- **Accent:** Cyan needle tip (`#00bcd4`) — clarity and focus
- **Inner ring:** White (`#ffffff`) — purity, inclusion
- **Grid:** Medium purple (`#7b1fa2`)

## Symbolism
- **Purple** = Creativity, wisdom, inclusivity
- **White ring** = Universal access, clarity
- **Cyan accent** = Focus, attention to detail
- **Wheel/compass** = Navigation for all abilities

## Emoji Mapping
♿ = Universal access / wheelchair accessibility
