# Tempofox Avatar Brief

**Concept:** Agile fox badge with warm amber palette signifying adaptable rhythm.

## Visual Notes
- Copper dusk gradient (#1a0c03 → #060201) with amber grid (#ffb347).
- Needle base sand (#ffd8a0) and tip coral (#ff7f50) leaning slightly west for agility.
- Highlight (#fff2d2) adds satin finish.

## Generator Flags
```bash
scripts/generate-avatar.sh \
  --output avatars/tempofox-avatar.png \
  --mode color \
  --bg-top '#1a0c03' \
  --bg-bottom '#060201' \
  --grid '#ffb347' \
  --needle-tip '#ff7f50' \
  --needle-base '#ffd8a0' \
  --tail-cutout '#0b0401' \
  --highlight '#fff2d2' \
  --needle-angle -4 \
  --grid-spacing 58
scripts/generate-avatar.sh --output avatars/tempofox-avatar-mono.png --mode mono --needle-angle -4 --grid-spacing 58
```
