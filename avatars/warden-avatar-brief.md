# Warden Avatar Brief

## Visual Concept
Shield-lock governance motif representing the control-plane governor.

## Design Elements
- **Primary:** Deep navy/indigo shield form (`#1a1a2e` to `#16213e`)
- **Accent:** Crimson lock symbol (`#e94560`)
- **Grid:** Subtle purple grid overlay (`#533483`)
- **Center:** Dark compass rose (`#0f3460`)

## Symbolism
- **Shield** = Protection of the lattice
- **Lock** = Enforced boundaries
- **Grid** = Structured governance
- **Compass** = Direction/navigation for agents
